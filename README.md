# ans-abs-template

Template for American Nuclear Society conference abstracts.


### Instructions
1. Update all ``FIXME`` items.
2. Building
- To build on a Linux machine run ``make``.
- To build on a Windows machine run
  - ``pdflatex main.tex``
  - ``bibtex main.tex``
  - ``pdflatex main.tex``
  - ``pdflatex main.tex``
3. To get rid of built files, ``make clean.``
